3. Questão 5.4.1 - 3

A função one_of permite que a função select selecione uma variável caso ela esteja dentro dos valores fornecidos. Para o vetor vars <- c("year", "month", "day", "dep_delay", "arr_delay") a função pode ser útil caso se quisermos selecionar as variáveis incluídas nesse vetor, desta forma select(flights, one_of(vars))

cancelled <- flights %>%
  group_by(year, month, day) %>%
  summarise( n = sum(is.na(dep_delay)))

cancelled <- flights %>%
  group_by(year, month, day) %>%
  summarise( n = sum(is.na(dep_delay)), delay = mean(dep_delay, na.rm = TRUE))

ggplot(cancelled, aes(delay, n)) +
  geom_smooth(se = FALSE)

Dias com atrasos médios maiores tiveram mais voos cancelados.
